# Desarrollo de una aplicación de busqueda de usuarios Github

El objetivo de esta prueba es diseñar e implementar una pequeña aplicación web de buscador de usuarios de GitHub utilizando las herramientas de tu elección.

## Contexto

En esta prueba, pretendemos desarrollar una aplicación web que permita a los usuarios buscar perfiles de GitHub y obtener información básica sobre ellos.

## API

- URL de la API de GitHub:

```
https://api.github.com/users/{username}
```

- URL de la API de los repositorios:

```
https://api.github.com/users/{username}/repos
```

## Requisitos

### Funcionalidad

1. **Búsqueda de Usuarios:** La aplicación debe permitir a los usuarios buscar perfiles de GitHub ingresando el nombre de usuario en un campo de búsqueda.

2. **Visualización de Perfil:** Una vez que el usuario realiza una búsqueda, la aplicación debe mostrar la información básica del perfil de GitHub, como el nombre de usuario, descripción, la foto de perfil, localización, número de seguidores y numero de seguidos.
